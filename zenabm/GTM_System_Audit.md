|-------------|-------|---------------|

### Where to Focus

**Identify your weakest layer:**

| Weakest Layer | Priority Actions |
|---------------|------------------|
| **BUILD is lowest** | Fix your foundation first. Clean up ICP, build proper segmentation, invest in data enrichment. |
| **ACTIVATE is lowest** | You have good targeting but poor execution. Add channels, coordinate timing, align sales + marketing. |
| **MEASURE is lowest** | You're running programs but can't prove they work. Build account scoring, implement holdout testing, move beyond leads. |
