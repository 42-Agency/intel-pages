Claude Code Cheatsheet for B2B Marketers

Installation (2 minutes)

# 1. Install Node.js from nodejs.org
# 2. Open Terminal and run:
npm install -g @anthropic-ai/claude-code
# 3. Start Claude Code:
claude
Run inside a project folder so Claude can read your files. Create a folder like ~/Projects/campaign-analysis and run claude there.

Prompts That Actually Work

Analyze Performance Data "Look at google_ads_export.csv. Which campaigns have CPL above $150? Show me the trend over the last 4 weeks."
Competitive Research "Go to [competitor.com], analyze their homepage messaging, pricing page, and any case studies. Summarize their positioning."
Content Repurposing "Turn this blog post into 5 LinkedIn posts. Each should have a hook, one key insight, and end with a question."
Build a Report "Create an HTML report from this CSV with charts showing spend by channel and CPL trends. Make it look professional."

Context = Better Output

  • Be specific: "Q1 vs Q4 performance" not "analyze this"
  • Share the goal: "This is for a QBR with the CMO"
  • Give examples: "Format like this: [paste example]"
  • Set constraints: "Keep it under 200 words"
  • Name your files: Drag files in or "look at data.csv"
If output is too generic: "Be specific. Use actual numbers from the data. Skip the caveats."

CLAUDE.md = Project Memory

Create a CLAUDE.md file in your project folder. Claude reads it automatically every session.

# Client: Acme Corp
B2B SaaS, Series B, $50k/mo ad spend

## What I'm working on
